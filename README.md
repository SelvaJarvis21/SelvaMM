# 💰 Selva MM (Money Manager)

A native Android budget-tracking application engineered to simplify personal expense management and visualize financial data through clean categorical breakdowns.

## 🚀 Key Features
* **Dual-Stream Ledger:** Seamlessly tracks both inbound financial income and outbound categorical expenses.
* **Dynamic Dashboard:** Aggregates raw transactional inputs to generate a real-time comparison of Expense vs. Income.
* **Categorical Analytics:** Automatically segments financial habits into distinct user-facing categories for rapid financial assessment.

## 💡 Engineering Insights
Building this application provided deep hands-on experience with:
* **State Management:** Keeping the financial dashboard synchronized accurately with real-time transactional inputs.
* **Data Flow:** Structuring relationships between user inputs, monetary mathematical operations, and frontend visual graphs.

## 🛠️ Tech Stack
* **IDE:** Android Studio
* **Language:** Kotlin
* **UI Architecture:** Android XML / Jetpack Compose
